# Release v1.0.0

这是公开仓库版本的首版说明，定位为可本地部署、可网页使用、可继续二次开发的 A 股智能分析与选股基础版本。

## 本版包含

- Web 工作台
- 问股分析页
- 收盘后选股页
- 沪 A 大盘情绪侧栏
- 多策略选股与自定义 YAML 策略
- 次日操作手册
- T+1 / T+3 / T+5 / T+10 回测
- 回测优化总结
- 飞书通知
- 多数据源回退与本地缓存

## 默认行为

- 选股范围固定为沪深主板、非 ST
- 每种策略最多保留 10 只候选股
- 默认仅对前 5 只生成详细报告
- 支持在交易日 16:00 后自动执行默认策略

## 数据源策略

- `TickFlow`：优先用于历史数据与实时行情
- `Tushare`：作为辅助和回退
- `Tavily / Bocha`：用于新闻检索与消息面补充
- `OpenAI 兼容模型 / DeepSeek`：用于分析与操作手册生成

## 仓库安全约定

本公开仓库不包含以下敏感内容：

- 真实 API Key
- `.env`
- 本地数据库
- 日志文件
- 缓存与运行时数据

请使用仓库中的 `.env.example` 自行填写本地配置。

## 已知建议

- 如果你的模型网关对非流式请求兼容性一般，建议保留回退模型
- 如果新闻源质量不稳定，建议至少配置两个搜索源
- 如果想减少接口频率限制影响，建议优先启用 TickFlow 并保留同日缓存
